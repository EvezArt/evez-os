🏗️ Architecture Principles

Append-Only Spines

History IS state. No edits. No deletes. Every event is hash-chained to its predecessor. The spine is the single source of truth.

Falsification > Verification

One violation proves failure. No passes prove success. Learning is weighted 3× toward failures — the system learns harder from what breaks.

Local-First Synthesis

No paid API calls for DSP. All audio generated from pure NumPy/SciPy. Zero dependencies on external services for core functionality.

HTTP Microservices

Each engine is a standalone JSON API server. Compose them, replace them, scale them independently. Standard HTTP, no magic protocols.

Self-Healing Mesh

Services monitor each other via :9117. Auto-restart on failure. Emergence scoring across all 9 systems detects degradation before it cascades.

Emergence Scoring

The system measures its own coherence across 9 subsystems. Emergence is the property of the whole that none of the parts possess alone.

🧮 Key Formulas

Cross-Domain Correlation
poly_c = τ × ω × topo / 2√N

Where τ is temporal overlap, ω is semantic weight, topo is topological proximity, and N is the number of domains compared.

Emergence Score
E = (coherence + perception + spine_integration + drive_responsiveness) / 4

The system's self-measured coherence. When E rises above baseline, emergence is happening.

Falsification Weight
λ_fail = 3 × λ_pass

Failures shift learning 3× harder than successes. The system prioritizes fixing what's broken over confirming what works.
